Om typeracers

typeracers eller skriv hastighets tests är ett typ av sätt att beräkna hur många ord du kan skriva per minut/sekund genom att mäta genom ett 1 minuter långt test. Din skriv hastighet kan vara bra att veta för att veta ungefär hur snabbt du kan arbeta på jobbet och hur effektiv du är. Det kan även vara en kul sak att bli bra på och jämnföra dig med med dina kompisar på hur snabbt dem kan skriva. Om du skapar ett konto hos oss så kommer all dina tidigare skriv hastigheter sparas så att du kan mäta om du blir bättre eller sämre på att skriva snabbt.
